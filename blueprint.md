# **Project: Lotto Number Recommender**

## **Overview**

A web application that generates and displays recommended lottery numbers for the user.

## **Project Outline**

*   **HTML (`index.html`):**
    *   Header with the title of the application.
    *   A main content area to display the generated lottery numbers.
    *   A button to trigger the number generation.
    *   Use of a custom element `<lotto-numbers>` to display the results.
*   **CSS (`style.css`):**
    *   Modern and visually appealing design.
    *   Responsive layout for mobile and web.
    *   Use of CSS variables for theming.
    *   Visually appealing styles for the number display and button.
    *   Subtle background texture.
    *   Drop shadows for depth.
*   **JavaScript (`main.js`):**
    *   Implementation of the lottery number generation logic (6 unique numbers from 1 to 45).
    *   A Web Component `LottoNumbers` to encapsulate the display of the numbers.
    *   Event listener for the "Generate" button.
